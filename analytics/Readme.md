#### ... my own analytical process definition is the next one:

***STEP 1:*** **IDENTIFY/DEFINE** the problem and **TYPE** of problem (enough)

***STEP 2:*** **APPROACH** How to solve? <br/>

SELECT:
- Tool <- Why?
- Technique <- Why?
- Method <- Why?

***STEP 3:*** **IMPLEMENT** <br/>
- Tool       
- Technique  
- Method

***STEP 4:*** **SOLVE + EVALUATE** <br/>
- OK? -> End
- KO? -> Check STEPS 1, 2 and 3
